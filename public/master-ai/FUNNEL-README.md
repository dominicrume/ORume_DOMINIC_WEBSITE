# Master AI in 9 Days — Landing Page (rumedominic.com/master-ai)

A keyword-optimized, conversion-focused lead-capture page for the free
"Master AI in 9 Days" course + the free book "From Code to Consciousness."
Feeds the Vorem value ladder (free course/book -> paid courses at vorem.co).

## Folder structure
```
site_build/
├── README.md              (this file)
├── ARCHITECTURE.md        (how the funnel + page fit together)
├── DEPLOY.md              (step-by-step: go live on rumedominic.com)
└── master-ai/
    ├── index.html         (the landing page)
    └── assets/            (put real testimonial photos / proof screenshots here)
```

## What's optimized
- **Keywords** (from live research): "learn AI free", "free AI course for beginners",
  "how to learn AI 2026", "AI for beginners", "no experience", "AI literacy",
  woven into the H1, title, meta, body, and FAQ.
- **SEO**: title tag, meta description, meta keywords, canonical URL.
- **AEO** (answer-engine optimization): FAQ section + FAQPage schema so AI engines
  can quote direct answers ("Can I learn AI for free?").
- **Social (X / sharing)**: Open Graph + Twitter Card tags.
- **Structured data**: Course schema + FAQPage schema (JSON-LD).
- **Conversion** (from 2026 CRO research): 5th-7th grade reading level, one promise,
  two-field form, "what happens next" transparency, real testimonial slots, star
  ratings, clear single CTA repeated, mobile-first.

## IMPORTANT — honest limits
- "GEO / Google Trends / Reddit / X / news optimization" are NOT page properties.
  They are (a) inputs that informed the keywords above, and (b) about WHERE you
  promote the page. No single HTML file can be "Reddit-optimized." Promotion is a
  separate activity (post the page where your audience already is).
- Fill testimonials with REAL quotes/screenshots only.
- Wire the form to Brevo (see DEPLOY.md) — the page alone can't store emails.
